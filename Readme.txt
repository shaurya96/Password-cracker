1) If using ubuntu hashcat can be installed by using command "sudo apt install hashcat" or if using windows we can install hashcat binaries by visiting website "https://hashcat.net/hashcat/".
2) For example - In my case I have used windows as it had more processing power to crack the required number of hashes. In windows ZIP file will be installed which has to be extracted.
3) The path where hashcat commands have to be executed is "C:\Users\hp\Downloads\hashcat-6.2.6\hashcat-6.2.6".
